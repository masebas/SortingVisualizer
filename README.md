# Sorting Visualizer
## Description
   A tool for visualizing different sorting algorithms.
   Current algorithms:
   - Insertion Sort
   - Merge Sort
   - Quick Sort

   Planned algorithms:
   - Heap Sort
   - Bubble Sort
   - Selection Sort
   - Counting Sort
   - Bucket Sort
   - Radix Sort


## Disclaimer
   This tool was not made with performance in mind, but purely as a way to visualize the process performed by different sorting algorithms.
